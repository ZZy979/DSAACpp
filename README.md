# DSAACpp
《数据结构、算法与应用C++语言描述》书中ADT代码

## 环境
Visual Studio 2019

## 目录结构
解决方案包含ADT和Tests两个项目
* ADT：包含所有数据结构类的代码
* Tests：ADT对应的单元测试

## 数据结构
ADT项目包含的数据结构类如下
* Linear_list：线性表
    * [Linear_list.h](ADT/Linear_list/Linear_list.h)：线性表抽象基类
    * [Array_list.h](ADT/Linear_list/Array_list.h)：线性表的数组实现
* [utils.h](ADT/utils.h)：辅助函数
